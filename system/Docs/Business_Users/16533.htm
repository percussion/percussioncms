Navigating with Keystrokes

Rhythmyx is accessible to users who navigate with keystrokes (screen reader application users or users whose system is otherwise unavailable for navigation with a mouse).

The default keystrokes for navigating in Rhythmyx Content Explorer are the following:

Keystroke

Function

CTRL+A

Select all items in main display panel

CTRL+C

Copy from clipboard

CTRL+V

Paste from clipboard

CTRL+B

Open About Rhythmyx dialog

CTRL+S

Open Search dialog

CTRL+0

Open Display Options dialog

CTRL+R

Refresh selected node in navigation pane

The default keystrokes for navigating and selecting options in dialogs and Content Editors are indicated by an underlined letter in a word beside the field control or option.

For example, the underlined C in Content indicates that ALT-C displays the Content menu. The underlined N in New indicates that ALT-N opens the Content > New menu. The underlined F in Folder indicates that ALT-F opens the Create Folder menu:

If the underlined letters do not appear in your browser, hold down ALT to display them. Continue to hold ALT down as you click the appropriate letters.
