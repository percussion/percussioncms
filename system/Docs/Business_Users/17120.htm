Accessing the Active Assembly Interface from an Action Menu

To access the Active Assembly interface from a Content Explorer Action Menu:

  1. In Content Explorer, right-click on a Content Item and choose Active Assembly. The Active Assembly option only appears on the Action Menu if the item is checked out by you or available to you to check out.

    The sub-menu displays the names of Templates available for the Content Item.

     

  2. Choose the name of a Template.

    Rhythmyx displays the Template in the Active Assembly interface.

For information on using the Active Assembly interface, see the online help available from the interface.
