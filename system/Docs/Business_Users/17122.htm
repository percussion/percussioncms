Accessing the Active Assembly Interface from the Site Explorer Menu

To access the Active Assembly interface from Content Explorer's Site Explorer drop menu:

  1. In Content Explorer, choose Site Explorer > [site name].

    Rhythmyx opens the Site's home page in the Active Assembly interface. From there, you can access the rest of the Site in Active Assembly.

For information on using the Active Assembly interface, see the online help available from the interface.
