Advanced Searches When Using the Full-text Search Engine

Like a simple search, an advanced search lets you enter a specific word or phrase from the content item that you are looking for in the Search for field. In addition, an advanced search allows you to define the specific fields you want to search and the values to search for in those fields. The default version of the Advanced search includes the following fields:

  • Title
  • Content ID
  • Community
  • Content Type
  • Created by
  • Start Date

Enter the values you want to use to search in the fields you want to search. For example, if you wanted to search for a specific content item and you knew its ID, you could enter the ID in the Content id field and the search engine would return that specific content item.

The field Expand query with synonyms is useful if you are looking for content items about a certain topic, but don't know the exact word they use to refer to it. For example, if you wanted to find content items that referred to your company's educational offerings, you might enter the term education in the Search for field and check Expand query with synonyms. The search would return content items that include words such as training and class as well as education.

You can also customize an advanced search.
