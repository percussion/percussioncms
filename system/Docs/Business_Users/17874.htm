Rhythmyx Full-text Search Engine

The Rhythmyx full-text search engine provides a very flexible capability for searching your repository for content items. It includes the feature Expand query with synonyms, which enables you to find content items that include terms similar in meaning to your Search for term.

When entering a Search for term into the Content Search dialog, use Apache's Lucene query syntax.
