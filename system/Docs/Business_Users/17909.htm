Fuzzy Searches

To do a fuzzy search use the tilde, "~", symbol at the end of a Single word Term. For example to search for a term similar in spelling to "roam" in Search for, enter:

roam~

This search will find terms like foam and roams.

You can also add a value after the tilde to indicate the degree of similarity in spelling you want to search for. The value must be between 0 and 1. Values closer to 0 indicate less similarity. Values closer to 1 indicate greater similarity.

For example:

roam~0.8

The default that is used if the value is not given is 0.5.
