Saving Search Queries

Each time you create and run a new search, Rhythmyx saves the search query in the Search Results node. If you do not run additional searches, the search query in Search Results is saved for the current session only; you may rerun it during the current session by clicking Search Results.

You can save your search query to make it available in future sessions and to other users.

If your system has migrated from the database search engine to the full-text search engine, any saved searches you created using the database search engine will still exist and will continue to use the database search engine functionality. To use the full-text search engine, you must create and save a new search query. Saved searches created while using the database search engine will continue to exist until you delete them manually.
