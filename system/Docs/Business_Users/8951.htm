Editing a Saved Search Query

To edit a saved search query:

  1. In the Navigation Tree, click the Search Query, and in the Menu Bar, select Content > [search name] > Edit Query.

    OR

    Right-click on the Search node and choose Edit Query in the drop menu.

    Rhythmyx displays the Content Search dialog filled in for the selected Search Query. If your system has been upgraded from the database search engine and you have not edited the search query since the upgrade, Rhythmyx displays the Advanced search dialog. Otherwise, Rhythmyx displays the Content Search in the format you used when you originally saved the query.

  2. You can change any of the values in in field, change the Mode of the search, convert it from Simple to Advanced or Advanced to Simple, or add or delete fields from an Advanced search.
  3. Click [OK].

Rhythmyx saves the changes to the Search Query.
