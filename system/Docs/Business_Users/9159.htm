Sending a Content Item's URL to Another User

If you want another user who has access to the Rhythmyx Server to view one of your Content Items, you can copy its URL in Rhythmyx and paste it into email or another application that the other user can access. Note: The application must be able to communicate with Rhythmyx and the user must have authorization to view the Content Item to access it.

To send a Content Item's URL to another user:

  1. Display the Content Item in Content Explorer.
  2. Right-click on the Content Item to display its Action Menu.
  3. Choose Copy URL to Clipboard.
  4. In the user's application, paste the URL into an appropriate control.
  5. The user may click on the URL to access the Content Item. A dialog prompts for username and password if the user has not yet logged in to Rhythmyx.

The URL is in the format:
http://<Rhythmyx host>:9992/Rhythmyx/sys_ActionPage/Panel.html?sys_contentid=<content ID#>
For example: http://10.10.10.10:9992/Rhythmyx/sys_ActionPage/Panel.html?sys_contentid=308

The host must be a specific name or IP address, and may not be localhost or 127.0.0.1.
