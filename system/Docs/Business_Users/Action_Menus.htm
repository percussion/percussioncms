Action Menus

In Rhythmyx, Action Menus let you take action on a Content Item. Action Menus are available in Content Explorer for the selected or active Content Item.

In Content Explorer, when you select a Content Item and right-click in the Display pane, Rhythmyx displays an Action Menu with some or all of the options displayed in the following graphics. Various factors, such as your security access, your Community, and whether or not the Content Item is in a public State, determine which options are available.

Action Menu for Content Item not in Public State

Action Menu for Content Item in Public State

  • The menu displays the Quick Edit option, which lets you remove an item from a public State for editing. This option is only available if the item is in a public State.
  • The menu displays an Edit option if you have the current item checked out or it is available for you to check out. If you select Edit but you do not have the item checked out, Rhythmyx automatically checks it out to you. Edit opens the Content Editor for the Content Item. The option is only available if the item is not in a public State.
  • The menu displays a View option which displays:
    • a Content link, which opens the Content Editor for the Content Item in View format (the fields cannot be edited);
    • a Properties link, which opens the Properties Editor for the Content Item;
    • a Compare link, which takes you to the Comparison Page for the Content Item;
    • Links to the Revisions and Audit Trail pages for the Content Item.

  • The Preview option displays links to all of the Template previews for the item:

  • The Workflow option displays:
    • A sub-menu with all of the Workflow Transitions available to you, including Check-in or Check-out for items that are not in a public State, and Force Check-in if you are in an Administrator role. Force Check-in lets an Administrator check in a Content Item checked out by another user. The Ignore Publish transition transitions an item for Quick Editing.

      NOTE: If your log-in Community is different from the Community of the Content Item, Check-in and Check-out are the only Workflow Transitions available to you.

  • The menu displays the Create option if the Content Item is checked in.

    The Create option displays:

    • a Create Translation option that opens a pop up where you can specify the properties of a Translation Copy and save it; only available if your Rhythmyx system has been globalized;
    • a New Copy option that lets you create a new Copy of the Content Item;
    • a Promotable Version option that lets you create a promotable Version of the Content Item. Only available if the item is in a public State.

  • The Active Assembly option displays links to all of the Templates of the item in Active Assembly mode.  The links take you to the Active Assembly interface. In Active Assembly, you can edit fields and manage content in Slots. Only available if the item is checked out to you or available to you for check out and it is not in a public State.

  • The Active Assembly for Documents displays links to all of the page Templates of the item. The links take you to the Document Assembly window for the Content Item.

    Only available if the item is checked out to you or available to you for check out and it is not in a public State.

  • The Impact Analysis option opens the Impact Analysis window for the Content Item.  
  • The Remove from Folder option removes the link to the Item in the selected Folder. Only available when a Folder is selected in the Navigation Pane.
  • The Copy URL to Clipboard option copies the URL of the Content Item to the clipboard so that the user can paste it into another application.
  • The Purge option is only available to users in Administrator roles; it lets an administrator delete a Content Item from the Rhythmyx repository.
  • The Copy option lets you copy a Content Item for a copy/paste operation.
