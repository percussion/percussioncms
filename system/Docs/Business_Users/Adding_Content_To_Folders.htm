Adding Content To Folders

If you have Admin or Write permission in a folder you can Copy a folder "from" it.If you have Admin or Write permission for a folder, you can create a new Copy of a Content Item in the folder or add an existing Content Item to the folder. No users have permission to add a Content Item to the main Folder node. Remember that Content Items are actually located in the Rhythmyx data repository, and the nodes in the folders are actually links to them.

To create new content in a folder, see Creating New Items In a Content Explorer Folder.

To create a new Copy of a Content Item or add an existing Content Item to a folder by dragging and dropping:

  1. Display the Content Item to copy in the Display pane by selecting a folder, view, or search in the Navigation Tree
  2. Drag the item from the Display pane and drop it into a folder in the Navigation Tree.

    Rhythmyx displays a menu of paste actions.

    OR

    Right-click the item in the Display pane and choose Copy in the Action Menu. Then right-click a folder in the Navigation Tree and choose Paste in the drop menu.

    Rhythmyx displays a sub-menu of paste actions.

  3. To move the Content Item to the folder and remove it from the original folder, choose Move.

    Rhythmyx inserts a node for the Content Item in the folder and removes the node for the Content Item from the original folder. The node is actually a link to the Content Item in the Rhythmyx data repository.

  4. To add the Content Item to the folder, and also leave it in the original folder, choose As Link.

    Rhythmyx inserts a node for the Content Item in the folder. The node is actually a link to the Content Item in the Rhythmyx data repository.

  5. To create a Copy of the Content Item in the folder, select As New Copy.

    Rhythmyx creates a new Copy of the Content Item with a new Content ID, and the folder displays a new node for it. The new node is actually a link to the new Content Item in the Rhythmyx data repository. You must open the Copy to edit it.
