Arranging Content in a Folder

Use the drag and drop functionality or Copy and Paste options in Content Explorer to move or create new copies of folders and their contents in the Navigation Tree. When you move a folder into another folder, Rhythmyx deletes the moved folder and its Content Item links from their original location and inserts them in the new location. When you copy a folder into another folder, Rhythmyx creates a new folder and new Copies of the Content Items in the folder.

To move or copy a folder and its contents into another folder by dragging and dropping:

  1. Display the folder that you want to move in the Display pane by selecting the node above the folder that contains it in the Navigation Tree.
  2. Drag the folder from the Display pane and drop it into a target folder in the Navigation Tree.

    Rhythmyx displays a drop menu with Paste options.

    Choose Paste > Move to move the folder and its contents.

    Rhythmyx moves the source folder and its Content Item links into the target folder and deletes the original folder and Content Item links.

  3. Choose Paste > As New Copy to copy the folder and create new Copies of its contents.

    Rhythmyx copies the source folder and creates Copies of its Content Item links and inserts them into the target folder. You most open the Copies to edit them.

To move or copy a folder and its contents into another folder using Copy and Paste options:

  1. Display the folder that you want to move in the Display pane by selecting the node above the folder that contains it in the Navigation Tree.
  2. Right-click the folder in the Display pane.

    Rhythmyx displays a drop menu.

  3. In the drop menu, select Copy.
  4. Right-click the target folder in the Navigation Tree.

    Rhythmyx displays a drop menu.

  5. Choose Paste > Move to move the folder and its contents.

    Rhythmyx moves the source folder and its Content Item links into the target folder and deletes the original folder and Content Item links.

  6. Choose Paste > As New Copy to copy the folder and create new Copies of its contents.

    Rhythmyx copies the source folder and creates Copies of its Content Items and inserts them into the target folder. You most open the Copies to edit them.
