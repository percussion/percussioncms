Content Explorer Interface

Content contribution, editing, and approvals are most often completed in the Content Explorer. To log into this web interface, a user must use a supported web browser and navigate to http://servername:port/Rhythmyx where servername:port describes the location of the Rhythmyx server. This information can be acquired from your Rhythmyx Administrator. An example might resemble: http://contentmanager:9992/Rhythmyx. When bookmarking this page, be sure the URL does not contain any information other than the servername:port and the word "Rhythmyx" with the first letter capitalized as shown in the example.

After you log in, Rhythmyx opens Content Explorer. The Content Explorer interface appears and functions similarly to the Windows Explorer interface: the left pane displays a Navigation Tree and the right pane is a Display pane that lists the contents of the node selected in the Navigation Tree. Content Explorer also includes a Menu Bar with drop menus, and a banner with user session properties.

Notice that an icon precedes each content item listed in the display pane. Depending on the way your system is configured, the same icon may represent all Content Types, or different icons may represent different Content Types, as in the following graphic.

The Navigation Tree has four main nodes:

  • Sites - Function like the Folders node, but Contains folder structures of sites. Provides you with a location for setting up a folder tree that resembles the structure of an external site or sites.
  • Folders - Contains folders that contain sub-folders or Content Items.
  • Views - Contains views for displaying Content Items. Each view displays content sorted and selected according to predefined parameters.
  • Searches - Contains search queries.

When you select a node in the Navigation Tree, the Display Pane displays it contents. You cannot view Content Item nodes in the Navigation Tree; you must select the folder, search, or view node that contains them in the Navigation Tree to view them in the Display Pane. In the Display Pane, you can select a Content Item node and right-click to open a menu of options for acting on the Content Item.

The folders, views, and searches in the Navigation Tree are virtual locations; the Content Items that they contain are actually located in the Rhythmyx data repository. The Content Item nodes in the folders, views, and searches are links to the actual Content Items. In general, you can think of the Content Items as being located in folders, views, and searches, but knowing that the nodes are actually links is important for understanding that you can insert the same Content Item nodes in multiple folders, views and searches.

In the Menu Bar, the Content menu displays options for working with folders, views, and saved searches in the Navigation Tree. You can access a menu of similar options by right-clicking on a node in the Navigation Tree.

The other Menus display options for refreshing the page, changing its appearance, accessing Site Explorer, and accessing Help. The menu options vary depending on the node selected in the Navigation Tree and your access and permissions in Rhythmyx.

At the top of Content Explorer, the Community and Locale entries link to pages that let you change these properties. In addition, a Logout button lets you log out of your session.

You may not have access or visibility to all of the Content Explorer options and functions described in the Help. Access is determined by several factors, including your Role, your Community, and the Content Type of each Content Item.

If you frequently log in to Rhythmyx, you may wish to bookmark the page in your browser. Once you have accessed the home page, Rhythmyx's URL is http://servername:port/Rhythmyx/sys_cx/mainpage.html. Bookmarking this URL can cause unexpected results when you switch users or Communities. Instead, bookmark the URL as http://servername:port/Rhythmyx.
