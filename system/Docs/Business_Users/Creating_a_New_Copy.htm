Creating a New Copy

You can create new copies of Content Items from the Action Menu if the Content Items are checked in.

A new copy has the community of the user who creates the new copy, not the community of the content item originator. If the Workflow of the copied Content Item is not in the Community of the user that creates the copy, the Workflow will be added to their Community.

To create a new copy of a content item from the Action Menu:

  1. In the Action Menu select Create > New Copy.

    Rhythmyx creates a new copy of the content item, assigns a unique ID to the new content item, and displays the new content item in Edit view in the Edit Content page.

  2. Edit the fields you want to modify for the new copy.  
  3. Click the [Update] button to save the new copy.  

    If you do not click [Update], Rhythmyx will save the new item but not the changes you made to it.

  4. Click [Close].

    Rhythmyx saves the new copy. You can find it in your Views, but not in the folder of the original Content Item. If you want to move it into the original folder, copy it from a Search or a View.

Another way of making a new copy is through the Copy and Paste > As New Copy option for adding content to a folder.
