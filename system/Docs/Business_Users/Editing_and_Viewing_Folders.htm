Editing and Viewing Folders

When you select Properties in a Folder's right-click menu, Rhythmyx displays the Folder Properties dialog. Your permission determines whether you can edit or view the fields in the Folder Properties dialog.

To edit or view folder properties:

  1. Right-click the folder in the Navigation Tree, and choose Properties in the menu.

    Rhythmyx opens the Folder Properties dialog to the General tab.

  2. If you have Read or Write access, you may only view the information on the General tab. If you have Admin access, you can modify the Folder Name, Folder Community, Description, Default Display Format, and Global Template fields on the General tab. You can also check or uncheck the Publish Only in Special Edition checkbox.
  3. If you have Read or Write access, you may only view the Folder ACL List information on the Security tab. If you have Admin access, you may modify the permissions of any entry under Names, you may add Names, and you may remove Names.
  4. If you have Read or Write access, you may only view the information on the Properties tab. If you have Admin access, you may modify the property Names, Values, and Descriptions.
