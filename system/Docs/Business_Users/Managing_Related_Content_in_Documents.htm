Managing Related Content in Documents

In the Document Assembly window, you can change the order, location, and Template of related content in a document.

To manage Related Content in documents:

  1. Expand the main Content Item node to display its Slots in the Navigation Tree and the selected Slot's related contents in the Display pane.
  2. To change the order of related Content Item in a Slot, drag and drop it among the other related Content Items in the Slot.

    Rhythmyx displays the Drop Actions dialog.

    Select Move > [Template] or As Link > [Template].

    If you choose As Link, Rhythmyx inserts an additional node for the Content Item in the Slot and leaves the original node; the item appears twice in the Slot. If you choose Move, Rhythmyx inserts a node for the Content Item in the Slot and deletes the original node; the item only appears once in the Slot.

    OR

    Right-click on the related Content Item, and select Arrange > Move Up/Left or Arrange > Move Down/Right in the Action Menu.

  3. To change the location, drag and drop it or copy and paste it from its current Slot to the Slot with the desired location. The Content Type and Template must be allowed in both Slots.
  4. To change a Content Item's Template within a Slot, right-click on the related Content Item, and select Change Template > [Template] in the Action Menu.
  5. To remove related content from a Slot, right-click on the related Content Item, and select Arrange > Remove in the Action Menu.
