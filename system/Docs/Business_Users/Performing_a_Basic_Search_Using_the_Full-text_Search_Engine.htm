Performing a Basic Search Using the Full-text Search Engine

To perform a basic content search:

  1. In the Navigation Tree, double-click the New Search node

    OR

    In the Menu Bar, choose Content > Search.

    OR

    Right-click in the Navigation Tree pane and choose Content > Search in the drop menu.

    OR

    To search in a specific Folder, right-click on the Folder in the Navigation pane and choose Content > Search from the popup menu.

    Rhythmyx displays the Content Search dialog.

  2. Enter the word or words or word fragment you want to search for in the Search for field. There is a limit of 100 characters in this field.
  3. Select the Display Format you want to use to display the results of the search.
  4. If you want to limit the number of results returned, uncheck the Unlimited box next to the Maximum Results field, and enter the maximum number of results you want to display in the Maximum Results field.
  5. The Include Subfolders checkbox is only included on this dialog if you initiated the search by right-clicking on a Folder in the Navigation pane. Check this box if you want to include Content Items in Subfolders of the selected Folder in your search. Uncheck the box if you want to limit the search to the specified Folder. The default is checked.
  6. Click the [OK] button to start the search.
