Searching for Content Items

You can define searches to Content Explorer to help you find content, and you can save these searches so you can use them again later. Your system may also include pre-defined searches created during system implementation. Unlike searches you define yourself, you won't be able to change the fields that are included in pre-defined searches. You can still can enter the data you wish into these searches, however.

Rhythmyx uses different icons to indicate pre-defined and user-defined searches:

Pre-defined search icon

User-defined search icon

Note that you may not be able to search on some fields. Most fields are enabled for searching, but in some implementations of Rhythmyx, certain fields are not available for searching. Contact your system administrator to find out whether any fields are restricted in your system.

Two different search engines are available for Rhythmyx. Your system will use one or the other.
