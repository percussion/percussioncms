How Relationships Acquire Site or Folder Properties

Active Assembly Relationships acquire the dependent item's Site and/or Folder properties if these properties are included in the Display Format of the search results when the dependent is included in a Slot on the Owner. This is accomplished in a slightly different way in the Active Assembly interface and in Active Assembly for Documents. In the Active Assembly Interface, you specifically check options to include the Site or Folder in the dependent item's URL when adding it to a Slot. See the Inserting Content in Slots section of the Active Assembly Interface help for a full explanation of how to check these options when adding Snippets to Slots.

In Active Assembly for Documents, you choose a display format for your search results that includes Site and/or Folder properties.
