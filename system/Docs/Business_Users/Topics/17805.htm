Expand Query with Synonyms

When Expand Query with Synonyms is checked, the search engine searches on words with similar meanings to the words you enter as well as matching the characters entered. For example, suppose you entered the phrase "job seeker" as a query. The search engine might return content items that include the words "applicant", "candidate", "hired", and "opening" as well as content items that include the phrase "job seeker".

Expand Query with Synonyms does not work properly if you enter special characters or wild cards. In a simple search, if you enter special characters or wild cards and check Expand Query with Synonyms, the search is not expanded to query with synonyms. In an advanced search, if you enter special characters or wild cards and check Expand Query with Synonyms, a warning indicates that the special characters are treated as word breaks.

By default, Expand Query with Synonyms is not checked.
