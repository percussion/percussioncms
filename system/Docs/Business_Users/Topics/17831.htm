Creating an Inline Link on an Inline Image

You may want to create a link from an inline image to another Content Item. Note that you can only create an inline link on an inline image, not on an image inserted as an inline Template (because the inline Template cannot be edited). To create an inline link on an inline image:

  1. Insert the image you want into your Content Item.
  2. Select the image.
  3. Click the inline link button .

    Rhythmyx displays the Content Browser.

  4. Browse or search for the content item to which you want to link and select an item.

    Only Page Templates are available for inline links.

  5. Choose the Template for the content item to which you want to link and click [Select].

    Rhythmyx creates a link on the image and closes the Content Browser. (NOTE: If the target Content Item is not Public, or if a link to the target page cannot be created for some other reason, no link will be published.)
