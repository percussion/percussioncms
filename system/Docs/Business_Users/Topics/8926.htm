Performing a Basic Search Using the Database Search Engine

To perform a basic search when using the database search engine:

  1. In the Navigation Tree, double-click the New Search node

    OR

    In the Menu Bar, choose Content > Search.

    OR

    Right-click in the Navigation Tree pane and choose Content > Search in the drop menu.

    Rhythmyx displays the Content Search dialog. None of the Search Criteria are required; you may enter values for any number of them.

  2. Enter the criteria for your search. For example:
    • Enter the Title of the Content Item you want to find.
    • Enter the Content ID of the Content Item you want to find.
    • Select the Community or Content Type of the Content Item you want to find.
    • Enter the State Date of the Content Item you want to find.

    You can enter multiple criteria. Adding more criteria to the search will make it more precise and more likely to return the results you are looking for.

  3. Select the Display Format you want to use to display the results of the search from the drop list.
  4. If you want to return a specific number of Content Items, uncheck the Unlimited checkbox and enter the number of results you want in the Max rows returned field.
  5. Click [OK] to launch the search.
