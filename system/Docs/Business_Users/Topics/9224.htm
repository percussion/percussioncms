Working with Rhythmyx Content Items Available in Web Folders

If WebDAV has been implemented in your system, you can access some Rhythmyx Content Items through Web Folders. Contact your System Administrator to find out whether WebDAV has been implemented in your system and to find out the URL of the WebDAV Folders to which you can connect.

To use a Web Folder, you must define a Network Place in Windows Explorer. To define a Network Place:

  1. In Windows Explorer, click on My Network Places.
  2. In My Network Places, double-click on Add Network Place.
  3. In the Type the Location of the Network Place field, enter the URL of the WebDAV Folder to which you want to connect (get the URL from your system administrator).

  4. Click the [Next] button.
  5. Windows Explorer will prompt you for your authentication information. Enter your Rhythmyx user name and password.
  6. Click [Finish] on the Completing the Add Network Place dialog.

Windows Explorer adds a new Network Place to your Network Places and displays the Web Folder to which you connected.

Once you have connected to a Web Folder, you can add new Content Items to Rhythmyx directly by dragging them into the Web Folder rather than by starting Rhythmyx. If you use another application that supports WebDAV, you can open Rhythmyx Content Items directly in that application without starting Rhythmyx (contact your system administrator to find out which applications support the WebDAV and how to configure your installation so you can access the Rhythmyx WebDAV-enabled Folders from that application. You can save any Content Item files you work on, or add new fields from your third-party applications as well.
