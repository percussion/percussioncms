Adding Related Content to a Document Slot by Dragging and Dropping

To create a new Copy of a Content Item or add an existing Content Item to a Slot by dragging and dropping in a Document Assembly window:

  1. Display the Content Item to copy in the Display pane by selecting its Slot in the Navigation Tree.
  2. Drag the item from the Display pane and drop it into the target Slot in the Navigation Tree.

    Rhythmyx displays the Drop Actions dialog.

  3. To add the Content Item to the Slot, select Move > [Template] or As Link > [Template].

    If you choose As Link, Rhythmyx inserts a node for the Content Item in the Slot and leaves it in the original slot. If you choose Move, Rhythmyx inserts a node for the Content Item in the Slot and deletes it from its original slot. The node is actually a link to the Content Item in its Rhythmyx directory.

For more information see Pasting: Links versus Copies.
