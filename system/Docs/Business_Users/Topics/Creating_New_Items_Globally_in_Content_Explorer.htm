Creating New Items Globally in Content Explorer

When you create a Content Item globally in Content Explorer, you create it without inserting it into a folder.

To create a Content Item globally in Content Explorer:

  1. Do not select any folders in the Navigation Tree.
  2. In the Menu Bar, select Content > New > Item > [Content Type].

    OR

    Right-click in the Navigation Tree pane and choose Content > New > Item > [Content Type] in the drop menu.

    Rhythmyx opens a Content Editor for the selected Content Type.

  3. Enter the Content Item data in the Content Editor.
