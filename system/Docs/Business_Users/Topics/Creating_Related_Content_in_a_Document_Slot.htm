Creating Related Content in a Document Slot

To add a new Content Item to a Slot in a Document Assembly window:

  1. Expand the main Content Item node in the Navigation Tree to display its Slots.
  2. Right-click the Slot in the Navigation Tree and select Create in the drop menu

    OR

    In the Navigation Tree, select the Slot, and in the Menu Bar, select Content > [Slot Type] > Create.

    Rhythmyx displays a Create Item dialog

  3. The Content Type drop list includes all Content Types available for the Slot. Select a Content Type.
  4. The Template drop list includes all Templates for the Content Type and Slot selected. Select a Template.
  5. Click [Create].
  6. Rhythmyx displays the Content Editor for the Content Type you selected.
  7. Create the Content Item using the Content Editor.

The Content Editor that Rhythmyx displays when you click [Create] only allows you to add content into fields on the Content Editor form.  It does not allow you to access a detail editor.  If the type of content that you are creating includes fields added in this manner, you must add them to the new Content Item after you have saved it.
