Creating a New Word-based Content Item

To create a new Word-based content item:

  1. In the Content Editor, click Launch Word.

    A Microsoft Word document opens behind the Rhythmyx window.

  2. Enter the content into the Microsoft Word document.  The content should include:
    • one paragraph with the name of the author, formatted with the ArticleAuthor style;
    • one paragraph with the Display Title of the content item, formatted with the ArticleDisplayTitle style;
    • a one-paragraph abstract of the article, formatted with the ArticleAbstract style;
    • one or more paragraphs of body text, formatted with the ArticleBody style.

    NOTE: The styles may have different names in your system. Check with your administrator.

  3. Save the content item.  In the File menu, choose Save to Rhythmyx or click .

    Rhythmyx displays a File Name dialog:

  4. Enter a name for the file and click [OK].

    Rhythmyx displays a confirmation dialog.

  5. If you click [Yes], Word closes and Rhythmyx displays the new Content Item as the single result of a Search. NOTE: Rhythmyx does not save the Word item into a folder. You must copy it into a folder from a Search or View.

    If you click [No], Word closes and Rhythmyx returns to the previous window.

    The Content Item is not automatically inserted into a Folder; you must manually copy it into any folders.
