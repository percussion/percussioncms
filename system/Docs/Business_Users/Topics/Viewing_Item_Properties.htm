Viewing Item Properties

When you select View > Properties in the Action Menu, Rhythmyx displays the content item's properties in the Properties page.

At the top of the Properties page is a Content link that takes you to the Content page for the content item.

To View Properties:

  1. Click the Action Menu and choose View > Properties.  

    Rhythmyx displays the Properties page for the content item. The editor lists the properties in two rows at the top of the screen.
