Working with Folders

Rhythmyx lets you create regular folders and Site folders in the Navigation Tree. In regular Folders, you can arrange Content Items in them in a way that is convenient to you. For example, you could create folders to store items created by you by their Content Type or by the content contributor who is working on them.

You arrange Site folders and their contents to duplicate the manner in which you want them published on your site.

The Content Explorer's Navigation Tree includes main Folder and Sites nodes that you can expand to display their folders and contents.

When we talk about folders in Rhythmyx, we are discussing both folders that appear under the Sites node and folders that appear under the Folders node.

Actions available for folders include creating new folders, creating new Content Items in folders, and rearranging Content Items and folders in the Navigation Tree. Depending on your access, you may be able to perform some or all of these actions.

You can select multiple folders and Content Items in the Display pane on which to perform an action; however only actions available to multiple nodes and common to all of the selected folders and items are available. To select more than one folder or Content Item node in the Display pane:

  • To select a range of nodes, click the top node that you want to select, then press [Shift] and click on the bottom node that you want to select.
  • To select several nodes that are not in a range, click the first node that you want to select, then press [CTRL] and click each additional node that you want to select.

Best practice is to include fewer than two hundred Content Items in any one Folder. If a Folder includes significantly more than two hundred Content Items, system performance will begin to degrade. If a folder contains more than one thousand Content Items, excessive memory usage may cause system instability.
