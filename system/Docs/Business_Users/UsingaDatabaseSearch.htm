Using a Database Search

When you click a Slot and choose the Insert > Snippet option or when you choose to insert an inline object in a rich text editor, the Content Browser dialog opens and gives you different options for finding a content item to insert.

The database search engine is a light-weight search engine that uses query functionality embedded in the database system This search engine lacks the ability to search fields that consist of large blocks of text.

To search for a content item using a database search:

  1. Click an Insert > Snippet action in Active Assembly or an insert inline object button in a rich text editor.

    Rhythmyx displays the Content Browser with the Sites tab selected.

  2. Click the Search tab.

    The Content Browser displays the database search.

  3. Enter values in any of the Search Parameters . For example:
    • Enter the Title of the Content Item you want to find.
    • Enter the Content ID of the Content Item you want to find.
    • Select the Community or Content Type of the Content Item you want to find.
    • Enter the State Date of the Content Item you want to find.

    You can enter multiple criteria. Adding more criteria to the search will make it more precise and more likely to return the results you are looking for.

  4. If you want to limit the number of results returned, enter the maximum number of results you want to display in the Maximum Results field. The entry -1 signifies unlimited results. Click [Unlimited] to re-enter this value in the Maximum Results field.
  5. If you want to use the item in more than one Site, check Show sites. An item in another Site can find this item anywhere in its original Site if you check this box. You may also check Show folders. An item in another Site can find this item in its specific Site folder if you check both boxes.
  6. Click [Search].

    The content area displays your search results. In the following graphic, both the Show sites and Show folders checkboxes were checked, so the content area displays the Site and Folder of each content item returned. Otherwise, the content area would not display these values.

  7. To reduce the number of results, enter a string in Filter by Name.
  8. Click on the icon for the item that you want to insert and click [Open]

    OR

    click the name of the item.

    The Content Browser displays the Template Selection window with all of the Content Type's Templates allowed for the location in which you want to insert it listed in the Select a Template box and the item formatted using the first Template in the Template Preview box.

  9. Click on the various Templates to preview their appearance.
  10. The Include site in the link and Include folder in the link checkboxes are disabled. Include site in the link is checked if you checked Show sites in the previous window. Include folder in the link is checked if you checked Show folders in the previous window.
  11. Once you have determined the Template that you want to use, click [Select]. Only single-select is supported; you cannot select multiple items to insert.

    If you are inserting an item in a Slot, the Content Browser closes the Template Selection window and displays the previous view of the Sites or Folder tab so you can insert additional items. If you are inserting an inline object in a rich text editor, the Template Selection window and the Content Browser close.

  12. If you inserted a Snippet and have returned to the list of search results, add another item or click [Back] to return to the search window. Search for another item or click [Close] to close the Content Browser.
