JNDI Datasource Configuration Dialog

Use the JNDI Datasource Configuration dialog to create new Datasource configurations and modify existing Datasource configurations.

To access the JNDI Datasource Configuration dialog:

  • On the JNDI subtab of the Datasources tab, click the [Add] button.
  • On the JNDI subtab of the Datasources tab, select the Datasource Configuration you want to edit and click the [Edit] tab.

Field Descriptions

Name Name for the Datasource configuration. The convention when naming datasources is to prepend the name with the string "jdbc/". To help conform to this convention, Rhythmyx automatically includes this string in the Name field. Add your name after the string. Spaces are not valid in datasource configuration names.

Driver Name of the JDBC driver definition the Datasource configuration will use. Options include all current driver definitions.

Server Definition of the server the Datasource connects to. May be a hostname name or a connect string. For example, to connect to an Oracle database server, you would enter a string like @myHost:1521:ORCL.

Userid User ID used to connect to the specified server.

Password Password used with the specified user.

Minimum Connections The minimum number of connections using this Datasource available in the connection pool. Defaults to 5 for a new Datasource configuration.

Maximum Connections The maximum number of connections using this Datasource available in the connection pool. Defaults to 100 for a new Datasource configuration.

Idle Timeout Amount of time, in minutes, that a connection can be idle before being closed.
