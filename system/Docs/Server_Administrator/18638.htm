Dynamically Retrieving Server Log Data

To dynamically retrieve information from XML application Log:

  1. Click the Logging tab. If the View sub-tab is not selected, select it.
  2. In the Type box, check each type of log entry that you want to retrieve.
  3. In the Date box, enter the Start and End dates of the log entries that you want to retrieve. You must include a value for HH:MM:SS as well as YYYYMMDD.

    NOTE: You cannot retrieve entries of deleted logs. To increase the number of days that the Server saves logs change the Keep last [nnn] days of Log setting in the Logging tab's Settings sub-tab.

  4. Click [Get it].

    The log results open in a separate window.
