Defining the User to Act as Part of the Operating System in Windows 2000

To define a user to act as part of the operating system in Windows 2003:

  1. Click the [Start] button, then select Settings > Control Panel.
  2. On the Control Panel, double-click Administrative tools.
  3. In Administrative Tools, double-click Local Security Policy.
  4. In Local Security Settings, double-click User Rights Assignment.

    Windows will display a list of User Rights policies.

  5. Double-click Act as part of the operating system.

    Windows will display the Local Security Policy Setting dialog.

  6. Click [Add].
  7. Select the user to whom you want to grant the privilege.
  8. Click [Add].
  9. Click [OK] to return to the Local Security Policy Setting dialog.
  10. Click [OK] to complete granting the user right.
