Rhythmyx Server Administrator

After logging in to the Rhythmyx Server Administrator, you can perform all administration tasks from it. The Server Administrator separates administration tasks into nine categories displayed by upper tabs:

Status to view details of servers and applications and to start and stop applications.

Settings to set session performance and page caching settings.

Datasources to set up the definition of the data Rhythmyx uses to connect to an RDBMS installation.

Logging to specify logging options and to perform and to retrieve information from log records.

Directory Services to create and maintain Directory Service connections.

Security to define security providers, roles, server ACLs, back-end credentials, and encryption.

Search to maintain the search configuration for the Full Text Search engine.

Monitor to issue commands from a remote server.

Content Explorer to configure the Java plugin version and screen refresh options in Content Explorer.
