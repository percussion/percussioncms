Starting Rhythmyx Server as a Daemon in a UNIX Environment

The Rhythmyx server daemon starts automatically when you start your system. If you need to restart the daemon after shutting it down, change to the Rhythmyx bin directory and enter the following:

./RhythmyxDaemon start <Rhythmyx root directory>

When you press <Enter>, Rhythmyx server will start as a daemon.
