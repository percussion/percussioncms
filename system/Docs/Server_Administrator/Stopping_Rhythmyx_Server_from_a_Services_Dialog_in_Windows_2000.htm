Stopping Rhythmyx Server from a Services Dialog in Windows 2000

To stop Rhythmyx server from the Services dialog in Windows 2003:

  1. Click the [Start] button, then select Settings > Control Panel > Administrative Tools. In Administrative Tools, double-click the Services icon.

    Windows will display the Services dialog.

  2. Double-click the Percussion Rhythmyx Service.

    Windows will display the Percussion Rhythmyx Service Properties dialog.

  3. Click [Stop].

    Windows will shut down the Rhythmyx server as a Windows service.
