Maintaining JNDI Datasource Configurations

A JNDI Datasource consists of the following data:

  • JDBC driver
  • DBMS server
  • authentication credentials
  • minimum and maximum connection limits for the database connection pool
  • idle timeout

The JNDI tab lists Datasource configurations.

Use the JNDI Datasource Configuration dialog to define new Datasource configurations and edit existing configurations.

You can create a new Datasource configuration or edit an existing configuration.

To delete a configuration, select the configuration you want to delete and click the [Delete] button. (NOTE: You cannot delete a configuration that is currently in use by a database connection.)
