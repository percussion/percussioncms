Adding a Database Driver Definition

To add a new Database Driver definition:

  1. On the Drivers subtab of the Datasource tab, click the [Add] button.

    The Server Administrator displays the Driver Configuration dialog.

  2. In the Driver field, enter a name for the Driver definition.
  3. In the Class field, enter the fully-qualified name of the driver class; for example: net.sourceforge.jtds.jdbc.Driver.
  4. In the Container Type Mapping drop list, choose the name of the dataset used to connect to your RDBMS version. Options include all currently-supported RDBMS versions.
  5. In the Database Dialog field, choose the dialect of your RDBMS. Options include all currently supported relational database management systems.
  6. Click the [OK] button to save your definition.
  7. On the Server Administrator, click the [Apply] button to apply your changes to the server.

You must restart the Rhythmyx server before your changes will take effect.
