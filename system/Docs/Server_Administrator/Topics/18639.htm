Configuring XML Application Logs

To configure XML application logs that are regularly stored in the system:

  1. Click the Logging tab. Then click the Settings tab.
  2. Check or uncheck any settings under Keep log of to configure the Server to log certain types of events.
  3. Check or uncheck Auto Delete to specify whether or not you want the system to automatically delete log entries after certain time intervals.
  4. If you check Auto Delete, enter a value in Keep last [nnn] days of Log to specify how many days the system should save a log entry before deleting it.
  5. Click [Apply] to save your settings or click [OK] to save your settings and close the Server Administrator.
