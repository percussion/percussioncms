Adding a Java Plugin Configuration

To add a Java Plugin Configuration:

  1. On the Rhythmyx Server Administrator, click the Content Explorer tab.
  2. Double-click in the first empty cell of the Client OS column and select the operating system for which you want to define the configuration.
  3. Double-click in the Browser cell of the same row and select the web browser for which you want to define the configuration.
  4. Double-click in the Plugin Version cell of the same row and select the Java Plugin Version you want to assign to the configuration. You can also enter the plugin version manually by clicking in the cell and entering the plugin Version, however the versions included by default have been tested and are known to work well in a variety of configurations.
  5. Double-click in the Versioning cell of the same row and select the Versioning option for the configuration.
  6. Double-click in the Plugin location cell of the same row and select the location from which you want users to download the plugin. The URLs defined in these options support Dynamic Versioning. You can also enter the location manually by clicking in the cell and entering a URL from which the plugin can be installed. You can use a Rhythmyx location (note that you must download the.cab from Sun to the specified location), or a location on the web. The URL to download plugins from Sun Microsystems is http://java.sun.com/products/plugin/. Click [Apply] to save the configuration. For the currently supported Java plugin, see the Rhythmyx readme or the Rhythmyx Concepts Guide.
