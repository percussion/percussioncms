Editing a Server ACL

To edit a Server ACL:

  1. Click the Security tab.
  2. Click the Server ACL tab.
  3. Click the entry you want to edit.
  4. Click the [Edit] button.  Rhythmyx displays the ACL entry details dialog.

  5. Edit the fields you wish to change and click [OK].
