Interfacing by remote console

You can issue the following pre-defined commands from a remote console to the Rhythmyx server and view results in HTML format.

To issue a command in the console interface:

  1. Click the Monitor tab and the Console tab.
  2. Type a command in the Command field. For details about the commands available, see Server Console Commands in Alphabetical Order.
  3. Click Execute.
