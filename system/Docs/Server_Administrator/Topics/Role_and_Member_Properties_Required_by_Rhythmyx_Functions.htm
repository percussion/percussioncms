Role and Member Properties Required by Rhythmyx Functions

Rhythmyx requires that you define role properties and member properties for some of its optional features. You can define properties when you configure a DBMS table security provider, or you can assign LDAP attributes as properties.

Role or Subject Property

Property Name

Function

Role

sys_defaultCommunity

Stores the name of the role's default Community. Required
when you are using the default Community feature.

Role

sys_defaulthomepageurl

Stores the fully qualified or relative URL of the first Rhythmyx
page that appears after login for users who log in under this role.
Required when you are using the default home page feature.

Subject

sys_email

Stores the email address of the login user. Required if you are using
the email notifications feature.
