Single Sign-on and the Rhythmyx Workbench and Server Administrator

When Rhythmyx is configured to support Single Sign-on, implementers can connect the Workbench to the server either directly or through the servlet, as shown in the following graphic:


The same option is also available to connect the Server Administrator to the server. Best practice, however, is to connect these clients in the same manner as the browser. If you prefer to connect these clients directly to the server when configured for Single Sign-on, you must define a security provider other than the Web server security provider. The Web server security provider cannot authenticate direct connections when set up to support Single Sign-on.
