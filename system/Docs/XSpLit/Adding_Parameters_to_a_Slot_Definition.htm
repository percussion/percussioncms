Adding Parameters to a Slot Definition

When using a remote template, you can add parameters to the template call as attributes, starting with parameter 1, then parameter 2, and so forth, as in the following example:

<span slotname="Navlink" template="NavlinkSlot" param1="'row-select', 'odd'">Navlink Slot Odd columns</span>

This code would produce the following XSL:

        <xsl:variable name="rxslot-1">

          <rxslot template="NavlinkSlot">

            <xsl:copy-of select="$related/linkurl[@slotname=&apos;Navlink&apos;]"/>

          </rxslot>

        </xsl:variable>

        <xsl:apply-templates select="$rxslot-1/*" mode="rxslot">

          <xsl:with-param name="row-select" select="&apos;odd&apos;"/>

        </xsl:apply-templates>
