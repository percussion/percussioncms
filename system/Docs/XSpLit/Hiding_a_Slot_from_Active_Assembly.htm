Hiding a Slot from Active Assembly

Rhythmyx automatically generates the content for some Slots, such as banners, navigation bars, and automated indices.  Because the user does not define the content for these Slots, they are not eligible for Active Assembly. To disable Active Assembly for the Slot, add the psxeditslot="no" attribute to the Slot definition.  The psxeditslot attribute can also take the value "yes", which enables active assembly in the Slot, but this is unnecessary.  Instead, omit the attribute, which enables Active Assembly by default.  

For example:

<span id="psx-hotarticles" slotname="Hot Articles" psxeditslot="no">Hot Articles here </span>

This markup produces the following XSL:

<xsl:variable name="rxslot-6">

<rxslot template="Hot Articles" psxeditslot ="no">

<xsl:copy-of select="$related/linkurl[@slotname=&apos;Hot Articles&apos;]"/></rxslot>

</xsl:variable>

<xsl:apply-templates select="$rxslot-6" mode="rxslot"/>

This Slot would not be eligible for Active Assembly.
