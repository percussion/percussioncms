Repeating Multiple Nodes

If the field has multiple nodes, it repeats once for each shared node. Any repeated node after a shared node is repeated in the same row. This technique is useful for grouping subordinate nodes together.

For example, if the field were psx-item/feature, and it were included in the following HTML code:

<table border="1" width="40%">
<tr>
<th bgcolor="lightgrey" align="left">Features</th>
</tr>
<tr>
<td align="center">psx-item/feature</td>
</tr>
</table>

The row would repeat once for each occurrence of item, since that is the first shared node. Any repeating occurrences of feature would be included in the same row.

Output: HTML

<feature>It's cheap!</feature>

<feature>It's fun!</feature>

Three occurrences of item, with varying occurrences of feature for each item would look like the following. Notice that the additional node item allows you to group the features together in rows.

Content: XML

Output: HTML

<features>
<item>
<feature>It's cheap!</feature>
</item>
<item>
<feature>It's easy!</feature>
<feature>It's fun!</feature>
</item>
<item>
<feature>Low maintenance</feature>
</item>
</features>
