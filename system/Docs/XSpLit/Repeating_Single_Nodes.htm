Repeating Single Nodes

If a field has a single node, the table row repeats once for each occurrence of the field in the run-time XML document.

Suppose you have the following HTML code:

<table border="1" width="40%">
<tr bgcolor="lightgrey">
<th align="left">Features</th>
</tr>
<tr>
<td align="center">psx-itemfeature</td>
</tr>
</table>

At design time, the code would produce this output:

At run time, the output would look like this for one occurrence of the field:

Content: XML

Output: HTML

<features>
<itemfeature>It's cheap!</itemfeature>
</features>

If there were more than one occurrence, the cell would repeat. In this example, the field repeats 3 times:

Content: XML

Output: HTML

<features>
  <itemfeature>It's cheap!</itemfeature>
  <itemfeature>It's easy!</itemfeature>
  <itemfeature>It's fun!</itemfeature>
</features>
