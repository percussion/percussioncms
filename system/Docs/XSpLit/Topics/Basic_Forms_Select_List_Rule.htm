Basic Forms Select List Rule

About the Root Node

In a Select list in a Form, each option element repeats based on the first node (going left to right) of the field listed in the Option tag.

For example, if you have the fields psx-product/number and psx-product/name in the following HTML code:

<form method="POST">
<p>
<select name="choices" size="1">
<option selected value="none">Select a product</option>
<option value="psx-product/number">psx-product/name</option>
</select>
<input type="submit" value="Go" name="Go">
</p>
</form>

Within the select tags, the option tag repeats once per product. The value of each option is the value of the products/product/name field. The value of the value attribute for each option is the value of the products/product/number field.
