Exposed Syntax-Fields as Tag Content

To use exposed syntax with tag content, specify the field name as the content of the tag.

HTML: <tagname>psx-fieldname</tagname>

XML: <fieldname></fieldname>

For example:

<p>psx-Name/First</p>

<td>psx-Product/Part/Number</td>

The field name must be the only content in the tag. If there is other content in the tag, or if you would not normally include the field name within existing tags, place the field inside the HTML <span> tag.

For example, the following code would simply display the psx-Date as part of the text, rather than replacing it with the dynamic data:

Incorrect:

<p>Today's date is psx-Date.</p>

In order to process the psx-Date field correctly, you must enclose it in the <Span> tag:

Correct:

<p>Today's date is <span>psx-Date</span></p>

Other examples:

<p>Hello<span>psx-Name/First</span>, today's date is <span>psx-Date</span>.</p>

<td> Part: <span>psx-Product/Part/Number</span> </td>

Note: You can use both the Exposed and Hidden syntax together in the same HTML page.
