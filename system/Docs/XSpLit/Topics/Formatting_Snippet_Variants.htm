Formatting Snippet Variants

Snippet Variants are parsed (processed as XML) when they are assembled into Slots. Therefore, Snippet Variants must generate completely well-formed output. All formatting on the Snippet must be well-formed HTML (or other XML markup), and non-well-formed areas must be encased within character data markers.

Snippet Variants must also include the the output method declaration: xsl:output method="xml". Place the declaration at the beginning of the page using the following code:

<!-- begin XSL -->

<xsl:output method="xml" omit-xml-declaration="yes" />

<!-- end XSL -->

If you do not include this declaration, the page is not tidied correctly as XHTL and Assembly will generate errors when merging the Snippet into a Slot.
