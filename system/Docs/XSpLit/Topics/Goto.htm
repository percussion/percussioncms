Goto

Use the Goto line dialog to jump to a specific line in the document you are viewing. To access the Goto line dialog,

choose Edit > Goto; or

press CTRL-G.

Enter the line number you want to jump to, and click [OK]. The cursor will jump to the line you specified. If you specify a value greater than the number of lines in the current file, the cursor jumps to the end of the file.

 

To close the Goto line dialog,

click [Cancel]; or

press <Escape>.
