Hidden Syntax

Use Hidden Syntax if you want a better idea of how dynamic data will appear in the HTML page at run time.

To use Hidden Syntax:

  1. Use sample or dummy data for both text and page properties (such as hyperlinks).
  2. Hide the field names inside normal HTML tags surrounding the dummy data.

At design time, pages with dummy data values closely resemble the look of the pages at runtime. It is more difficult to see the fields, however, unless you look at the HTML source code.

You can use both Exposed and Hidden Syntax together on the same HTML page.

Note: The dummy data is included in the sample XML file created by XSpLit.
