Setting the Related Content Variable

Related Content calls use the variable $related.  XSpLit defines this variable at the top of each template that contains related content.  The variable definition is:

<variable name="related" select="/*/sys_AssemblerInfo/RelatedContent"/>

If you write a template manually, be sure to include this variable definition at the top of the template XSL file.
