Shortcut Using the Image Tag

The following examples show the use of the ID attribute shortcut in the IMAGE tag:

Example 1:

IMAGE:

<img src="sampleimageURL" id="psx-fieldname"/>

The field named in id="psx-fieldname" replaces the value of the 'src' attribute at run time.

 XSpLit-generated XML:

<fieldname>sampleimageURL</fieldname>

Example 2:

IMAGE:

<img src="placeholder.gif" id="psx-person/image"/>

The value of person/image replaces the 'src' attribute at run time.

 XSpLit-generated XML:

<person>

<image>placeholder.gif</image>

</person>
