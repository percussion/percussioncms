Special Shortcut Syntax

When using Hidden Syntax in IMAGE (IMG), ANCHOR (A), and FORM (FORM) tags, you can use the ID attribute of the tag as a shortcut to specify the dynamic data.

<tagname attribute ="value" id=psx-fieldname>

Note: If you put more than one id attribute in a tag, XSpLit uses only the first id attribute.
