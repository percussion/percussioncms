Specifying a Template for a Slot

In some cases, you may want to specify a template for a slot rather than defining it locally.  In that case, add the template attribute to the slot definition:

<span slotname="slotname" template="templatename"

For example:

<div slotname="Related Features" template="RelatedFeatureSlot">

<table width="100%" border="1">

<tr>

<td>Related Feature Slot</td>

</tr>

</table>

</div>

When split, this code would produce the following XSL:

<xsl:variable name="rxslot-1">

<rxslot template="Related-Feature Slot">

<xsl:copy-of select="$related/linkurl[@slotname=&apos;Related Feature&apos;]"/>

</rxslot>

</xsl:variable>

<xsl:apply-templates select="$rxslot-1" mode="rxslot"/>
