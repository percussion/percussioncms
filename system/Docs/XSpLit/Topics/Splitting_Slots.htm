Splitting Slots

A slot definition consists of a related content definition and slot formatting.  The formatting may be defined in the HTML, or you can call a template to provide the formatting.

All Slots in Rhythmyx are defined in one of two libraries.

  • sys_Slots.xsl stores all default Rhythmyx templates provided by Percussion Software.  This library may be overwritten when Rhythmyx is upgraded, so users should not modify the templates defined in this library.  This file located in the root/sys_resources/stylesheets/assemblers directory.
  • rx_Slots.xsl stores all user-defined templates.  It is not overwritten when Rhythmyx is upgraded.  This file is located in the root/rx_resources/stylesheets/assemblers directory.

When an HTML file that includes one or more Slots is split, Rhythmyx imports these two libraries, and any other Slot libraries that may have been defined.
