XSpLit Menu Commands

The following table describes XSpLit commands. You issue XSpLit commands from pull-down menus and/or toolbar buttons.

Button

Menu

Submenu

Task

File

New

Clear the content of all panes and insert the default document in Source HTML.

 

Open

Open the standard file selector dialog. (The Files of type field shows the file extension for the currently active pane.)

 

Save As

Save the contents of the currently active pane.

 

Save All As

Save all checked file types; you can save all panes simultaneously.

 

 

Exit

Exit XSpLit.

Edit

Undo

Undo the last action in the current tab.

 

Redo

Redo the last undone action in the current tab.

 

Cut

Cut the selected text and copy it to the system clipboard.

 

Copy

Copy the contents from the system clipboard to the current cursor position.

 

Paste

Copy the selected text to the system clipboard.

 

Delete

Delete the selected text.

 

 

Select All

Select all text in the current tab.

 

Find...

Displays the Find dialog.

 

 

Find Next

Finds the next occurrence of the current find settings.

 

 

Replace...

Displays the Find and Replace dialog.

 

 

Goto

Displays the Goto dialog.

 

View

 

Toggle the visibility of panes. By default, all panes are visible. (For more information, see XSpLit Tabs).

Tools

Split

Convert the Source HTML document to well-formed HTML, and generate a DTD, XSL and sample XML files.

 

Merge

Combine XSL and XML into an HTML document.

 

Launch Browser

Launch Browser

 

 

Options

Set parameters:

Enter an Identifying tag for HTML markup. (Default is psx-.)

Input file character encoding - When importing files from a system with different encoding, select UTF-8 or cp 1252.

Output file character encoding - When exporting files to a system with different encoding, select UTF-8 or cp 1252.

 

Help

Percussion XSpLit Help

Launch online help for the Splitter.

 

 

About XSpLit

Launch the About dialog, displaying version, copyright, and link to the Percussion home page.
