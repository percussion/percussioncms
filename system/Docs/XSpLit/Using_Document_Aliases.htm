Using Document Aliases

To use an alias in your HTML markup, enter it as a node name in a dynamic field name:

psx-/Aliasname/node

The alias refers to the URL defined for the alias in the HTML document header.

In a sample HTML file using document aliases, we find code that defines the alias gizmos:

 <!-- psx-docalias="gizmos" psx-docref="gizmo/gizmos.xml" -->

This alias is used to bring the content gizmo.xml into the HTML document. Another alias, gadget, brings a different content into the document. The result at design time looks like this:
